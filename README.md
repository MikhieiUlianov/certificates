# My Certificates & Credentials Registry

Welcome! This repository serves as a centralized, public registry for my professional certifications, verified technical credentials, and academic achievements. 

The purpose of this repository is to provide a single, transparent source of truth where employers, recruiters, and collaborators can view and verify my continuous learning journey, technical skill sets, and professional qualifications.

---

## What's Inside This Repository?

This repository contains high-resolution copies (images and PDFs) of my earned certificates, organized logically by category:

* **Cloud & DevOps:** Infrastructure, CI/CD pipelines, and cloud platform certifications.
* **Software Engineering:** Programming languages, frameworks, and system architecture credentials.
* **Methodologies:** Agile, Scrum, or project management qualifications.

---

## How to Verify My Credentials

1. **Browse the Assets:** You can navigate through the folders above to view the physical certificates directly inside GitHub.
2. **Official Verification:** For official verification, look for the direct validation links or credential IDs provided alongside the assets, which link directly to issuing platforms like Credly, Accredible, or university portals.
